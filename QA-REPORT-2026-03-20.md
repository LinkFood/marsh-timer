### P2 — Minor Code Quality

1. **hunt-log `'weather:unknown'` fallback still embedded** (Low severity)
   - **File:** `supabase/functions/hunt-log/index.ts:109`
   - **Issue:** When no weather record exists in `hunt_weather_history` for the given state/date, the string `'weather:unknown'` is embedded into hunt_knowledge. This pollutes the vector space with non-informative tokens.
   - **Fix:** Omit the weather component entirely when unavailable: `const weatherStr = weather ? \`weather:temp:...\` : '';`
   - **Impact:** Low — only affects hunt logs for dates/states without weather history. Core fix (using real data when available) is working.
