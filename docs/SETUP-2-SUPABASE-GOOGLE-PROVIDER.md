# Task 2: Enable Google Auth Provider in Supabase

## Where
Supabase Dashboard: https://supabase.com/dashboard/project/rvhyotvklfowklzjahdd/auth/providers

## Steps

1. Go to the Supabase Dashboard link above
2. Find **Google** in the list of auth providers
3. Toggle it **ON** (enabled)
4. Paste the **Client ID** from Task 1 into the Client ID field
5. Paste the **Client Secret** from Task 1 into the Client Secret field
6. Leave all other settings as defaults
7. Click **Save**

## Verification
The Google provider row should show as enabled (green toggle).
