## Key Strategic Insight
Most lucrative use cases (insurance, commodity trading, energy) require B2B sales team.
Most accessible use cases (weather nerds, date explorer, argument settler) are free/cheap but drive traffic + brand.
**Strategy: viral consumer features prove the brain, B2B API/reports are the business.**

## Top 5 by Revenue Potential
1. Insurance catastrophe analyst — $500-2K/mo/seat, 6-18mo sales cycle
2. Ag commodity trader — $200-1K/mo, proven market, data already exists
3. Energy trader — $1-5K/mo, competing with internal build
4. Event insurance underwriter — $200-500/report, self-serve possible
5. Legal expert witness — $50-200/report, 400K+ relevant cases/yr

## Top 5 by Audience Size
1. Weather obsessives / amateur mets — millions, daily engagement
2. Allergy sufferers — 50M Americans, premium ad CPMs
3. Fishermen — 50M licensed anglers
4. Climate anxiety demographic — tens of millions, high SEO value
5. Hunters — 15M, existing core audience

## Top 3 Viral Plays
1. Date Explorer / birthday card — personal, shareable, chain-reaction loop
2. Argument settler — comparison content shared in group chats
3. "Is it getting worse here?" state pages — Reddit/Twitter/local news viral

## Critical Gap Identified
State-level resolution limits consumer use cases (fishing, allergies, real estate, farming).
County-level resolution unlocks the monetizable tier.
