## Context
James asked for fundamental rethinking of DCD interface. 7M entries, self-grading loop, cross-domain collisions — all hidden behind a ChatGPT-style chat box. James is a day trader who reads charts. Current product makes him read essays.

## Core Thesis
STOP TRANSLATING DATA INTO WORDS. Show data as data. Charts, bars, heatmaps, diffs, receipts, thermometers, tapes, ledgers.

## Top 5 (highest conviction)
1. **The Tape** — continuous real-time one-liner feed. Brain is already talking, make it visible.
2. **Conviction Charts** — state as stock, plotted over time. Trader's native language.
3. **The Autopsy Table** — make grading the most dramatic, visible thing on the site.
4. **The Receipt** — structured data response instead of paragraphs.
5. **Inverse Search** — condition-first temporal query ("when did this happen before?")
