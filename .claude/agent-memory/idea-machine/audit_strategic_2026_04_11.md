## Core Finding
James's insight: "The real find out is it's actually finding yourself." The brain's self-knowledge IS the company. 204 graded claims, accuracy by domain/state/window, known blind spots — this is what no competitor can replicate (time-dependent moat). Making this visible = the Brain Report Card page.

## Three Moves
1. Brain Report Card at /accuracy (today) — makes the moat visible
2. Narrative density blitz — 10x correlation engine, from 2,200 to 20,000+ narrative entries
3. Forensic date report generator — first revenue, $200/report, 400K cases/yr TAM

## Frontend Reality
4 dead product visions still in codebase: Workbench (DeckLayout/panels), Alert Dashboard (CollisionFeed/FusionPanel/RegimeDetector), Chat Brain (NowPage), Weather Trigger (current). Only the last is the product. Kill the rest.

## Revenue Ranking
1. API billing (1 week, $10-50M TAM, high fit)
2. Date Card / Birthday Report (1 week, viral + $5-20M TAM)
3. Forensic weather report ($80M TAM, 2-3 weeks, Daubert-ready via accuracy page)
4. Crop insurance ($200M TAM, 1-2 months, needs county resolution)

## Narrative Layer = Feature Not Bug
Pattern links only form between narrative entries because language IS the universal schema. Do NOT try to make raw data link directly. Lean into narrative density. Correlation engine + compound-risk-alert + narrator are the most important crons.

## Chat Assessment
Chat is the weakest surface: 10-30s latency, $0.005-0.02/call, unstructured text. Date portrait returns 56 structured entries in <2s. Structured data > AI prose for every revenue use case.
