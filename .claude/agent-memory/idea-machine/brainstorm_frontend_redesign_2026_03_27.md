## Context
Intelligence Page exists with 3-col layout, FusionWeb SVG, 50-state scoreboard, arc narratives, outcome windows, live feed, track record. But the fusion moment (what connected to what), the grading loop (was it right?), and historical odds are invisible. James wants to read DCD like he reads ES/NQ charts.

## 10 Main Ideas

1. **Conviction Meter** — vertical thermometer next to score showing historical hit rate for this specific pattern fingerprint. Gray/amber/green zones. Uses precedent_accuracy + calibration data.
2. **Domain Collision Timeline** — 72hr horizontal timeline per state, domains as rows, colored segments for active periods, vertical collision lines where 2+ overlap. Gantt chart for convergence.
3. **Autopsy Drawer** — slide-out for graded arcs showing "Expected vs Actual" two-column layout. Green/red/amber color coding. Opus reasoning at bottom. Trade journal for the brain.
4. **Pressure Differential Board** — scatter plot replacing ranked list. X=score, Y=3-day delta. Top-right=hot+accelerating. Dots sized by domain count, colored by arc phase.
5. **Signal Autopilot Strip** — 40px aviation PFD strip per state. Speed tape (score+brackets), attitude bar (domain proportions), altitude tape (accuracy). Stack 5 to scan.
6. **Ghost Layer** — map layer showing graded false alarms (red) and misses (blue) as translucent circles. Reveals geographic blind spots over time.
7. **Handshake Counter** — Geiger counter showing rate of pattern_links/hour near brain heartbeat. Optional audio ticks. "Is the brain excited?"
8. **Split Verdict** — binary sports-betting widget for outcome-phase arcs. CONFIRM bar fills left, MISS countdown fills right. Grading as a visual race.
9. **Regime Detector** — national breadth indicator above ticker. Active/quiet regime. Horizontal bar showing today vs 90-day range. Market breadth for environment.
10. **Brain's Bookmarks** — importance-ranked (not time-ranked) knowledge entries. Most-referenced, most-confirmed embeddings. The brain's "most cited papers."
