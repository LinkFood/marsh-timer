## Context
213K data points, 22 content types, 14 crons — but the frontend is 6 static cards, a sortable table, and a text ticker. The data is a river and the UI is a dixie cup.

## 15 Ideas Generated

1. **Sparkline Everything** — 7-day inline sparklines behind every number. SVG polyline, no library.
2. **Convergence Decomposition Chart** — Stacked area chart showing component breakdown over 30 days per state. The "oh shit" chart.
3. **Signal Waterfall** — Vertical timeline with cross-referenced events linked by connector lines when multiple data types hit same state in 6hr window.
4. **Brain Activity Monitor** — 32px EKG-style bar showing vector searches, embeddings, cron fires per hour. The "alive" signal.
5. **Heatmap Matrix** — 50 states x 22 data types grid, colored by signal strength/recency. Pure Bloomberg density.
6. **"Last Time This Happened" Replay** — Historical pattern match cards showing similar past conditions + outcomes.
7. **Flyway Pulse Lines** — Animated lines along 4 flyways on map, pulsing with migration activity.
8. **State Deep-Dive Panel** — Full stock-profile page per state with all brain data, sparklines, recall.
9. **Morning Briefing Screen** — 5-second animated intel brief on app open. Cinematic scout report.
10. **Component Radar Charts** — Tiny inline spider charts per state in screener showing component fingerprint.
11. **Comparison Overlay** — Shift-click two states for side-by-side breakdown with verdict.
12. **Data Freshness Heartbeat** — 14 cron status dots, pulsing when they fire. "Is this thing on?"
13. **Pattern Matcher Feed** — Brain's internal cross-referencing results surfaced as system log.
14. **Species Heat Toggle** — Switching species changes which components are emphasized in all views.
15. **Notification Stream as Ambient Background** — Translucent data flowing across screen behind all tabs.
