## The Problem
James: "I know what we're trying to build. The problem is that we're trying to build something that's never been built before, so I don't have words to describe it."

## Core Synthesis
What DCD is = an **artificial hippocampus for the Earth**. Functionally identical to episodic memory in the human brain:
- Specific events stored with context
- Cross-referenced by temporal proximity
- Retrievable by similarity (vector search = hippocampal pattern completion)
- Self-consolidating over time (grading loop = memory consolidation)
- Says "this reminds me of something..." and pulls up the analog

Not a model (has facts, not weights). Not a search engine (notices autonomously). Not a prediction service (recognizes, doesn't forecast). Not a dashboard (discovers unknowns, doesn't display knowns).

- **Wayback Machine for Earth** — except this one LEARNS.

## Medium, Not Product
Key insight: DCD isn't a product in a known category. It's a new MEDIUM — like how photography wasn't "better painting" and spreadsheets weren't "digital ledgers." The gap between "what exists" and "what words exist to describe it" is the signature of a genuinely new medium.
