# DCD Shareable Artifacts brainstorm (2026-07-02)

Owner's problem: "good engine, but how we show it is hard." Lens = shareable artifacts that carry the engine (receipts baked INTO the artifact, not as strippable caption).

Core insight: the smallest honest unit = ONE graded verdict + its matched-control denominator. Make THAT detachable/travelable/un-strippable. Not a prettier dashboard.

Existing bones to reuse: `/date/YYYY-MM-DD` + `/court` shareable URLs, middleware.ts (runs on every path, can inject OG meta), useClaimFires/useClaims, CountdownClock, PrecedentCard, TheRecord postmortem text.

The 7:
1. **Verdict Card (canvas->PNG)** — Low, NO DEP. HIT/MISS card with `8.2% vs 8.0% control · n=4,874` baked in pixels. Save button on /court verdicts.
2. **Cascade Strip** — Med, no dep (canvas/SVG). The one proven thing (drought/ocean/bird-silence led July 2026 heat wave 7-11 days) as horizontal timeline image. The thesis in one glance.
3. **"Day you were born" OG image** — Med, **NEEDS @vercel/og or satori** (crawlers reject SVG/canvas). Middleware injects per-date OG tags. Birthday share magnet -> lands on real archive page.
4. **Cite the Archive** — Low, NO DEP. Copy-paste citation block reframes app as a citable SOURCE. Receipt IS the format.
5. **Today Badge (embeddable iframe)** — Med, no dep. `/embed/verdict` `/embed/today?state=XX` lives on forums/blogs, can't lie (shows misses).
6. **Tombstone Card** — Low, no dep. NOBODY-WOULD-THINK-OF: obituary card for the KILLED convergence index. Bragging about your own dead flagship = the trust moat made portable.
7. **Prove Me Wrong** — Med, dep for auto-unfurl. Live fire challenge link, countdown + pre-registered hypothesis; same URL flips to verdict when window closes.

James lens fit: trader-brain, receipts-first, show-don't-predict, denominators mandatory. Tombstone (#6) and Cite (#4) most novel. Note: DCD is PAUSED (Lupa is flagship 2026-07) — this was a requested brainstorm, not active work.
