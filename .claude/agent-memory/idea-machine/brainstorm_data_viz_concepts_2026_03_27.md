## Context
Challenge: 2.4M-entry brain does incredible cross-domain pattern recognition, self-grading, anomaly detection, correlation discovery — but it's all invisible. The owner is a day trader who wants to read environmental intelligence the way he reads ES/NQ charts: fixed positions, visual encoding over text, pattern recognition at a glance.

Existing visualizations: FusionWeb SVG (radial domain map), sparklines in rankings, component bar charts (ArcConvergence), arc timeline dots, fingerprint text cards, accuracy bars (TrackRecord), MultiSpark 30-day chart, METAR event list, signal feed.

## 10 Concepts (All Net-New)

### Country Level (2-second scan)
1. **50-State Pulse Grid** — 5x10 grid of 36x28px micro-cells. Background intensity = score, border style = arc state, 20px sparkline inside, colored dots for grade history, orange triangle for anomaly. Sorted by score. See entire country as visual texture.
2. **Pressure Map** — US map or grid colored by convergence RATE OF CHANGE (3-day slope). Diverging color scale: blue=dropping, white=stable, red=rising. Flash animation when crossing 65 threshold. See regional trends instantly.
3. **Anomaly Constellation** — Scatter plot: X=convergence score, Y=anomaly z-score. 4 quadrants: top-right = unprecedented convergence (watch), top-left = weird but not converging (early warning), bottom-right = familiar pattern, bottom-left = boring. Dots colored by arc state.

### System Level (5-second understanding)
4. **Correlation Web** — 8 domain nodes in circle. Arcs between domains = active correlations. Arc thickness = number of states with that correlation. Shows brain's DISCOVERED relationships system-wide, not per-state.
5. **Grading Ledger** — Time & Sales tape. Narrow vertical strip, each row = one grade. Colored border, state abbr, convergence bar, icon. Running accuracy counter at top. Visual texture tells if brain is learning (red at bottom, green at top).

### State Level (10-second deep dive)
6. **Signal Chain Diagram** — Left-to-right DAG showing causal path: raw data nodes -> pattern match nodes -> convergence components. Line thickness = similarity. FUNNEL shape = multi-source convergence (good). Single line = single-source dependency (weak).
7. **Confidence Thermometer** — Vertical 4px mercury gauge showing calibrated TRUST, not score. Fill = weighted accuracy of contributing sources. Tick marks = individual source accuracy. Crossed-out ticks = suppressed sources (<40%).
8. **Arc Heartbeat Strip** — Horizontal EKG waveform. Amplitude = intelligence density per hour. Vertical markers at act transitions. Past arc grade dots at left edge. Current portion pulses live. Shows brain's ATTENTION rhythm.
9. **Domain Contribution Waterfall** — Single horizontal bar segmented by domain contribution, sorted largest-first. Reference lines at 65 (elevated) and 75 (critical). Tiny carets for up/down from yesterday. Shows COMPOSITION, not 8 independent facts.
10. **Fingerprint Comparison Overlay** — Current 8-component bars overlaid on ghosted historical match bars. Alignment gaps visible. Border color = historical outcome grade. Shows "this looks like a pattern the brain got RIGHT (or wrong)."
