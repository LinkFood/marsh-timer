# First 60 Seconds Brainstorm — 2026-07-02

**Context:** Owner's stated problem — "we have a good engine but how we show it is going to be hard." Lens = the opening frame, the one interaction that teaches everything. Current live pages: ExplorerLanding (date/state picker + weather/bird/anomaly lines + Court strip + archaeology + ask-the-archive), CourtPage (docket, denominators, lift, "The Record" convergence postmortem), DatePage (±14-day archaeology strip + domain cards + "tell the story" LLM button + precedents).

**Verified engine facts to dramatize:** 7.6M cross-domain entries, decades deep, state-resolved, embedded. Self-grading claim court (matched controls, lift + denominators). Proven lane: coupled layers (drought/ocean/soil/birds/search-trends) LED the July 2026 heat wave by 7-11 days while thermometers were silent — birds went silent ~11 days early. Doctrine: SHOW don't predict, always show denominators, convergence scores are DEAD (proven noise, killed).

## The 7 ideas
1. **Birthday Cold Open** (Low, buildable today) — hero = one pre-focused date input, "Pick a day you remember. See what the Earth remembers." Routes to existing DatePage synthesis. Personal memory vs engine memory. STRONGEST front door.
2. **The 11-Day Rewind** (Low-Med) — scrubber/autoplay replaying the July 2026 heat wave cascade, heat arriving LAST after drought/SST/bird-silence. Hardcode the 11-day sequence as a const, animate stacked SVG sparklines (existing Sparkline pattern). The thesis as a 10-sec animation. THE PROOF SHOT.
3. **The Cold Type-Out** (Low, buildable today) — near-black load, one real anomalous sentence types itself out from already-fetched layerItems/anomaly data, then page assembles around it. Teaches "notices weird cross-domain things" in one line.
4. **Guess The Weirder Day** (Med) — two dateless real archive days, "which was stranger?", reveal denominators after guess. Smuggles the base-rate/self-grading lesson inside a game. Shareable.
5. **The Absence Meter** (Med) — NOBODY-WOULD-THINK-OF-IT. Hero visual = SILENCE not activity. Ghosted "expected" band, actual reading dropping through the floor, the gap shaded/labeled. Absence led the heat wave. Every other dashboard shows activity up; leading with the void is novel + ominous. Feeds off absence-detector cron.
6. **The Receipt Printer** (Low, buildable today) — latest Court verdict PRINTS like a dot-matrix receipt on load (mono, perforated edge, HIT/MISS + denominators + lift). Honesty flex made literal.
7. **Sonify the Cascade** (Med) — UNHINGED WILDCARD. Speaker on the Rewind; 11 days play as sound (drought drone, SST pad, bird-silence note CUTS OUT, heatwave = the drop). Web Audio API, no lib. First env-data product you LISTEN to.

**Recommended combo:** #1 (front door) + #2 (proof) + #6 (honesty), all ~buildable today no new deps/backend. #5 = sleeper novelty.

**Half-formed leftovers to riff later:** "two dates that rhyme" echo-finder; live "engine just noticed something" toast on real scanBrainOnWrite hits; full-bleed single-number cold open (whole screen = today's anomaly sigma pulsing).
